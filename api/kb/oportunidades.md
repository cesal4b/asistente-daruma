MÓDULO: OPORTUNIDADES

Resumen:
Registro y seguimiento de oportunidades de mejora. Ciclo: Registro → Priorización (obligatoria) → Gestión/Plan de Acción → Cierre.

Campos críticos al registrar:
- Descripción (OBLIGATORIO)
- Responsable (OBLIGATORIO)
- Unidad y Proceso destino

Flujo rápido:
1. Ir a Listado de Módulos → Oportunidades → Nuevo.
2. Completar campos obligatorios y Guardar.
3. Priorizar desde el listado (menú 3 puntos → Priorizar) asignando Impacto y Urgencia.
4. Generar Plan de Acción si aplica (menú 3 puntos → Generar plan de acción).

Advertencias:
- La priorización es obligatoria; sin ella el progreso se bloquea.
- Use Entrenamiento para pruebas.
