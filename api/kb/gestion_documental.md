MÓDULO: GESTIÓN DOCUMENTAL

Resumen:
Ciclo de vida: Solicitud → Configuración → Elaboración → Revisión → Publicación/Aprobación. Roles: Administrador General, Elaborador, Revisor, Aprobador.

Pasos rápidos:
1. Solicitudes → Nuevo (describir detalladamente el documento requerido).
2. Administrador: Pro Documental → Nuevo → completar metadatos (no modificar código/versiones automáticas).
3. Elaboración en editor HTML nativo → Guardar frecuentemente → Firmar y avanzar.
4. Revisión: usar sugerencias y comentarios; devolver si aplica.
5. Publicación: configurar visibilidad (Institucional recomendado) → Publicar.

Advertencias:
- No modificar campos de Código, Versión ni Fecha de vigencia manualmente.
