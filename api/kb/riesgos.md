MÓDULO: RIESGOS

Resumen:
Gestión del riesgo con etapas: Configuración → Identificación → Análisis → Controles → Aprobación → Supervisión. Defina equipos y contexto operativo antes de crear riesgos.

Pasos esenciales:
1. Riesgos → Definición de equipos → Nuevo (asigne roles).
2. Crear Contexto Operativo (ruta: Riesgos → Contextos → Nuevo).
3. Identificar riesgos desde el contexto → Nuevo Riesgo.
4. Registrar controles, evaluarlos y aprobar.

Notas:
- Use variables reutilizables para causas y consecuencias (Elemento de Análisis).
- Registre siempre lo que hizo en cada paso para trazabilidad.
