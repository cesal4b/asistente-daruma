MÓDULO: PLANES DE ACCIÓN

Resumen:
Flujo secuencial: Creación → Aprobación → Ejecución/Seguimiento → Cierre. Roles clave: Líder del Plan, Encargado de Aprobar, Responsable de la Acción, Responsable de Seguimiento, Responsable de Efectividad.

Puntos clave:
- El origen del plan (Hallazgos/Oportunidades/Riesgos) debe coincidir con el tipo de plan.
- Las acciones deben sumar 100% en ponderación.
- Al marcar 100% una acción, se finaliza y se pierde edición; subir evidencias antes de cerrar.

Pasos rápidos:
1. Listado de Módulos → Planes de acción → Nuevo.
2. Completar datos (tipo, proceso, responsables) → Guardar.
3. Crear acciones y asignar responsables → Solicitar aprobación.
