MÓDULO: AUDITORÍAS

Resumen:
Arquitectura: Programa de Auditoría → Auditorías. Regla: crear el Programa antes de cualquier Auditoría. Roles: Auditor Líder, Auditor, Auditado.

Pasos rápidos para auditoría interna:
1. Crear Programa → Programar auditoría dentro del programa.
2. Planeación (definir parámetros) → Aprobar plan.
3. Crear lista de chequeo → Ejecutar sesiones y registrar hallazgos.
4. Auditado acepta hallazgos → Generar planes de acción si aplica.
5. Evaluación y conclusión → Cerrar (no reversible).

Notas:
- Para auditorías externas siga flujo simplificado de 4 etapas.
- Colores de línea de tiempo indican estado (verde, rojo, azul).
