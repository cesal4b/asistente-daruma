MÓDULO: PROCESOS

Resumen:
El módulo "Procesos" es un espacio de consulta (solo lectura) que contiene la arquitectura de 3 niveles: Macroprocesos → Procesos → Procedimientos. Se navega desde Portal → Mapa de procesos → Arquitectura de macroprocesos. Cada bloque del mapa es interactivo y enlaza a la última versión aprobada. Los diagramas se visualizan y descargan vía Bizagi. Antes de abrir diagramas, revisar la trazabilidad (quién elaboró/revisó/aprobó).

Pasos rápidos:
1. Iniciar sesión → barra lateral izquierda → Portal → Mapa de procesos.
2. Hacer clic en un macroproceso para descender al nivel 2 y luego al 3.
3. Revisar la ficha del proceso (panel izquierdo) y el diagrama (panel derecho).
4. Descargar si necesita copia local (no modifica el repositorio central).

Advertencias:
- Use siempre la instancia de ENTRENAMIENTO para pruebas.
- El control de versiones es automático; no edite documentos descargados pensando que afectan al repositorio central.
