# Daruma — Oportunidades de Mejora

Ciclo de vida (4 fases): **Registro → Priorización (obligatoria) → Gestión y Plan de Acción → Cierre**.

## Registro
Acceso: Listado de Módulos → Oportunidades → Nuevo. **Use siempre Entrenamiento, nunca Producción.**

Campos obligatorios: **Descripción** y **Responsable** (advertencia: si asigna a otra persona, usted pierde el control del ejercicio). Opcionales: Impacto, Justificación, Presupuesto. La Oportunidad FODA se omite por ahora.

## Guardado y trazabilidad
Al guardar, la oportunidad pasa al listado de indexación (ID, nombre, tipo, origen). El historial inferior registra usuario, acción, estado, fecha y hora.

## Priorización (obligatoria)
Sin ella, el progreso se bloquea. Seleccione la oportunidad → menú 3 puntos → Priorizar → asigne Impacto y Urgencia (Bajo/Medio/Alto). El sistema calcula la prioridad automáticamente.

## Gestión de estado
Estados: Inactivo, Abierta, En Proceso, Cerrada, Rechazada. Para generar un Plan de Acción: menú 3 puntos → Generar plan de acción → tipo, proceso, área, descripción → Guardar. Cambie el estado a "En Proceso" al conectar con un plan.

## Cierre
Gestionar oportunidad → estado Cerrada → Justificación → Guardar. Queda en modo solo lectura.
